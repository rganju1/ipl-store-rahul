# IPL Jersey Store 🏏

A simple interactive web app to select IPL teams, choose players, and customize jersey name and number.

## 🚀 Live Demo
https://ipl-store-rahul.vercel.app

---

## 📌 Features

- Select IPL Team
- Choose Player from dropdown
- Auto-fill jersey number
- Customize name and number
- Live preview of jersey text

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
